## dlx

Provision lxd containers for development

### Synopsis

dlx provisions lxd containers for local development.

```
dlx [flags]
```

### Options

```
  -a, --address string   LXD Daemon uri (url or socket) (default "https://10.0.1.109:8443")
  -h, --help             help for dlx
  -v, --verbose          verbose logging
```
